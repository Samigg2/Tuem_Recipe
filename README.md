# 🍽️ TuemRecipe – Share the Vibe, Share the Food

**TuemRecipe** is a community-driven food sharing platform where home cooks, food lovers, and kitchen experimenters come together to share their favorite dishes, secret family recipes, and culinary creations.

Whether you're whipping up a quick weeknight dinner or perfecting your grandma's signature stew, TuemRecipe is where your food finds its people.

---

## ✨ What's the Deal?

- **Share Your Recipes** – Post your own dishes with ingredients, steps, and photos
- **Discover New Flavors** – Explore recipes shared by real people, not just chefs
- **Save Your Faves** – Bookmark recipes you wanna try later
- **Connect with Foodies** – Comment, like, and vibe with other home cooks
- **No Gatekeeping** – From street food to Sunday roasts, everything belongs here

---

## 🛠️ Built With

- **Frontend:** Vue 3 + Vite + TypeScript
- **API Layer:** GraphQL + Hasura (instant real-time APIs)
- **Database:** PostgreSQL
- **Backend:** Go for custom services
- **Container:** Docker

---

## 🚀 Getting Started

### Prerequisites
- Docker and Docker Compose
- Node.js 18+
- Go 1.21+

### 1. Clone it
```bash
git clone https://github.com/yourusername/tuemrecipe.git
cd tuemrecipe
```

### 2. Fire it up with Docker
```bash
docker-compose up -d
```

### 3. Set up Hasura
- Go to `http://localhost:8080`
- Track your tables
- Done.

### 4. Run the frontend
```bash
cd frontend
npm install
npm run dev
```

Visit `http://localhost:5173` and you're live.
